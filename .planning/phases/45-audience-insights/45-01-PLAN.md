- platform: 'all' | specific platform (default: all)
- period: '7d' | '30d' | '90d' (default: 30d)

Response structure matches AudienceInsights interface.

**Implementation:**
1. Auth check using verifyToken (same pattern as unified metrics)
2. Get user's connected platforms from PlatformConnection
3. For each connected platform:
   - Call platform service to get audience data (mock for now, real APIs in 45-02)
   - Aggregate demographics, behavior data
4. Calculate aggregated totals across platforms
5. Return structured response

- Gender: Male, Female, Other
- Locations: Top 10 cities/countries
