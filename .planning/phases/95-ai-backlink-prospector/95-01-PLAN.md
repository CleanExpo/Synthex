# Phase 95-01 Plan — AI Backlink Prospector: Service Layer + Prisma Models + API Routes

**Phase:** 95
**Plan:** 1 of 2
**Status:** Pending
**Created:** 2026-03-11

## Overview

Build the service layer for the AI Backlink Prospector. This includes Prisma models for persisting prospects and analysis results, a TypeScript service layer for opportunity discovery (Google Custom Search API) and domain scoring (OpenPageRank API), email templates for outreach, and REST API routes for CRUD operations.

## Data Sources

- **Google Custom Search JSON API** (100 queries/day free) — find resource pages, guest post opportunities, broken link targets
- **OpenPageRank.com API** (100 requests/day free, no auth required for basic use) — domain authority scores
- **Heuristic scoring** — fallback when API quota exhausted

## Prisma Models

### BacklinkProspect

```prisma
model BacklinkProspect {
  id              String   @id @default(cuid())
  userId          String
  orgId           String
  targetUrl       String
  targetDomain    String
  domainAuthority Int?
  pageRank        Float?
  opportunityType String   // resource-page | guest-post | broken-link | competitor-link | journalist-mention
  status          String   @default("identified") // identified | contacted | published | rejected
  pitchSent       Boolean  @default(false)
  pitchResponse   String?
  outreachEmail   String?
  notes           String?  @db.Text
  discoveredAt    DateTime @default(now())
  contactedAt     DateTime?
  publishedAt     DateTime?
  createdAt       DateTime @default(now())
  updatedAt       DateTime @updatedAt

  user            User     @relation(fields: [userId], references: [id])

  @@index([userId])
  @@index([orgId])
  @@index([opportunityType])
  @@index([status])
}
```

## Service Layer Files

### lib/backlinks/types.ts
All TypeScript interfaces:
- `BacklinkOpportunityType` — union type
- `BacklinkStatus` — union type
- `BacklinkProspectInput` — create payload
- `ScoredDomain` — OpenPageRank result
- `SearchOpportunity` — raw Google CSE result
- `OutreachTemplate` — email template structure
- `AnalysisResult` — full analysis result

### lib/backlinks/domain-scorer.ts
OpenPageRank API integration:
- `scoreDomain(domain: string): Promise<ScoredDomain>` — fetch DA/PR from OpenPageRank
- `scoreDomains(domains: string[]): Promise<ScoredDomain[]>` — batch (max 100)
- `heuristicScore(domain: string): number` — fallback: score by TLD (.edu=85, .gov=90, .org=60, etc.)

OpenPageRank API: `GET https://openpagerank.com/api/v1.0/getPageRank?domains[0]=example.com`
Header: `API-OPR: {OPEN_PAGE_RANK_API_KEY}`
Response: `{ response: [{ domain, page_rank_integer, rank, status_code }] }`

### lib/backlinks/opportunity-finder.ts
Google Custom Search API integration:
- `findResourcePages(topic: string, userDomain: string): Promise<SearchOpportunity[]>`
- `findGuestPostSites(topic: string, userDomain: string): Promise<SearchOpportunity[]>`
- `findBrokenLinkTargets(topic: string): Promise<SearchOpportunity[]>`
- `findCompetitorLinkSources(competitorDomain: string): Promise<SearchOpportunity[]>`
- `findJournalistMentions(brandName: string, topic: string): Promise<SearchOpportunity[]>`

Query strategies:
- Resource pages: `"resource" OR "useful links" "${topic}" site:*.edu OR site:*.org`
- Guest posts: `"write for us" "${topic}" -site:${userDomain}`
- Broken links: `"${topic}" "404" OR "page not found" "resources"`
- Competitor links: `link:${competitorDomain} -site:${competitorDomain}`
- Journalist mentions: `"${brandName}" "${topic}" site:*.com`

Google CSE API: `GET https://www.googleapis.com/customsearch/v1?key={GOOGLE_CSE_KEY}&cx={GOOGLE_CSE_ID}&q=...`

### lib/backlinks/outreach-templates.ts
4 email templates by opportunity type:
- `resourcePage(prospect, brand): OutreachDraft`
- `guestPost(prospect, brand): OutreachDraft`
- `brokenLink(prospect, brand): OutreachDraft`
- `journalistMention(prospect, brand): OutreachDraft`
- `getTemplate(type: BacklinkOpportunityType, prospect, brand): OutreachDraft`

### lib/backlinks/backlink-analyzer.ts
Main orchestrator:
- `analyzeOpportunities(params: AnalyzeParams): Promise<AnalysisResult>`
  - Calls opportunity-finder for all 4 types
  - Calls domain-scorer for each found domain
  - Filters by DA threshold (min 20)
  - Deduplicates
  - Sorts by domain authority desc
  - Returns top 50 prospects

## API Routes

### GET /api/backlinks/prospects
- Auth: verifyToken / getUserIdFromRequest
- Query params: orgId?, status?, opportunityType?, limit?, offset?
- Returns: `{ prospects: BacklinkProspect[], total: number }`

### POST /api/backlinks/prospects
- Body: BacklinkProspectInput (orgId, targetUrl, targetDomain, opportunityType, ...)
- Validates with Zod
- Creates BacklinkProspect record
- Returns: `{ prospect: BacklinkProspect }`

### POST /api/backlinks/analyze
- Body: `{ orgId, topic, userDomain, competitorDomains? }`
- Calls backlink-analyzer
- Saves BacklinkAnalysis record
- Bulk-creates BacklinkProspect records
- Returns: `{ analysis: BacklinkAnalysis, prospects: BacklinkProspect[] }`

### GET /api/backlinks/analysis
- Query params: orgId?, limit?, offset?
- Returns: `{ analyses: BacklinkAnalysis[], total: number }`

### POST /api/backlinks/outreach
- Body: `{ prospectId, orgId, outreachEmail, pitchResponse? }`
- Updates BacklinkProspect status to 'contacted', sets contactedAt, pitchSent=true
- Returns: `{ prospect: BacklinkProspect }`
