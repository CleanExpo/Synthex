**Remove Express ecosystem from dependencies (all imports in src/ legacy only, zero in app/ or lib/):**

- `express-rate-limit`
- `express-session`
- `express-validator`

If ANY import is found in app/ or lib/, keep that package and document why.

**Update lib/video/capture-service.ts:**
Add a try-catch dynamic import at the top so the module doesn't crash if puppeteer is unavailable:
```typescript
let puppeteer: typeof import('puppeteer') | null = null;
try {
  puppeteer = await import('puppeteer');
} catch {
  // puppeteer not available in production/serverless
}
```
Adjust the class to check `if (!puppeteer) throw new Error('Puppeteer not available in this environment')` at the start of capture methods.
