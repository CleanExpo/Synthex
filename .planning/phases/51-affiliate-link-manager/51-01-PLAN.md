# 51-01 Plan: Affiliate Link Manager Backend

## Objective

Build backend infrastructure for affiliate link management including data models, service layer, API routes, and React hook.

## Execution Context

- Follow patterns from Phase 48-50 (revenue, ROI, sponsors)
- Use Decimal type for revenue/commission amounts
- Integrate with existing RevenueEntry model (source: 'affiliate')
- Standard API route pattern with verifyToken auth

## Context

### Existing Models to Reference
- `RevenueEntry` - has source field supporting 'affiliate'
- `LinkBioLink` - has click tracking pattern
- `Sponsor/SponsorDeal` - hierarchical data model pattern

### New Models
- `AffiliateNetwork` - configured affiliate networks (Amazon, ShareASale, etc.)
- `AffiliateLink` - individual tracked links with network association
- `AffiliateLinkClick` - click events for analytics

## Tasks

### Task 1: Add Prisma models

Add three models to `prisma/schema.prisma`:

```prisma
model AffiliateNetwork {
  id            String    @id @default(cuid())
  userId        String

  // Network details
  name          String    // Display name
  slug          String    // amazon, shareasale, cj, impact, etc.
  apiKey        String?   // Encrypted API key
  trackingId    String?   // User's tracking/affiliate ID

  // Settings
  isActive      Boolean   @default(true)
  commissionRate Decimal? @db.Decimal(5, 2) // Default commission %

  createdAt     DateTime  @default(now())
  updatedAt     DateTime  @updatedAt

  user          User      @relation(fields: [userId], references: [id], onDelete: Cascade)
  links         AffiliateLink[]

  @@unique([userId, slug])
  @@index([userId])
  @@map("affiliate_networks")
}

model AffiliateLink {
  id            String    @id @default(cuid())
  userId        String
  networkId     String?

  // Link details
  name          String    // Display name
  originalUrl   String    // Original product/page URL
  affiliateUrl  String    // URL with tracking params
  shortCode     String?   @unique // For link cloaking

  // Tracking
  clickCount    Int       @default(0)
  conversionCount Int     @default(0)
  totalRevenue  Decimal   @default(0) @db.Decimal(12, 2)

  // Metadata
  productName   String?
  productImage  String?
  category      String?
  tags          String[]

  // Auto-insertion settings
  autoInsert    Boolean   @default(false)
  keywords      String[]  // Trigger keywords for auto-insertion

  isActive      Boolean   @default(true)
  createdAt     DateTime  @default(now())
  updatedAt     DateTime  @updatedAt

  user          User      @relation(fields: [userId], references: [id], onDelete: Cascade)
  network       AffiliateNetwork? @relation(fields: [networkId], references: [id], onDelete: SetNull)
  clicks        AffiliateLinkClick[]

  @@index([userId])
  @@index([networkId])
  @@index([shortCode])
  @@index([autoInsert, keywords])
  @@map("affiliate_links")
}

model AffiliateLinkClick {
  id            String    @id @default(cuid())
  linkId        String

  // Click data
  ipHash        String?   // Hashed IP for dedup
  userAgent     String?
  referer       String?
  country       String?

  // Conversion
  converted     Boolean   @default(false)
  revenue       Decimal?  @db.Decimal(12, 2)
  orderId       String?

  createdAt     DateTime  @default(now())

  link          AffiliateLink @relation(fields: [linkId], references: [id], onDelete: Cascade)

  @@index([linkId])
  @@index([linkId, createdAt])
  @@index([createdAt])
  @@map("affiliate_link_clicks")
}
```

Add User relations for new models.

Run `npx prisma db push` to apply schema.

**Commit**: `feat(51-01): add affiliate link prisma models`

### Task 2: Create AffiliateLinkService

Create `lib/affiliates/affiliate-link-service.ts` with:

**Types & Constants**:
```typescript
export type NetworkSlug = 'amazon' | 'shareasale' | 'cj' | 'impact' | 'rakuten' | 'custom';
export const NETWORK_SLUGS: NetworkSlug[] = ['amazon', 'shareasale', 'cj', 'impact', 'rakuten', 'custom'];
export const NETWORK_LABELS: Record<NetworkSlug, string> = {...};
```

**Network Methods**:
- `listNetworks(userId)` - Get user's configured networks
- `getNetwork(userId, networkId)` - Single network
- `createNetwork(userId, data)` - Add network config
- `updateNetwork(userId, networkId, data)` - Update network
- `deleteNetwork(userId, networkId)` - Remove network

**Link Methods**:
- `listLinks(userId, filters?)` - Filter by network, category, active
- `getLink(userId, linkId)` - Single link with click stats
- `createLink(userId, data)` - Add affiliate link
- `updateLink(userId, linkId, data)` - Update link
- `deleteLink(userId, linkId)` - Remove link
- `generateShortCode()` - Create unique short code

**Tracking Methods**:
- `trackClick(linkId, metadata)` - Record click event
- `recordConversion(linkId, data)` - Record conversion with revenue
- `getLinkAnalytics(userId, linkId, dateRange)` - Click/conversion stats

**Auto-insertion**:
- `findMatchingLinks(userId, content)` - Find links matching keywords in content
- `insertAffiliateLinks(userId, content, options)` - Replace keywords with affiliate links

**Aggregation**:
- `getAffiliateStats(userId, dateRange?)` - Total clicks, conversions, revenue by network

~300 lines following sponsor-service.ts patterns.

### Task 6: Create useAffiliateLinks hook

Create `hooks/useAffiliateLinks.ts`:

```typescript
interface UseAffiliateLinksOptions {
  networkId?: string;
  category?: string;
  activeOnly?: boolean;
}

interface UseAffiliateLinksReturn {
  // Data
  networks: AffiliateNetwork[];
  links: AffiliateLink[];
  stats: AffiliateStats;
  isLoading: boolean;
  error: string | null;
  isMutating: boolean;

  // Actions
  refetch: () => Promise<void>;

  // Network mutations
  createNetwork: (data: CreateNetworkInput) => Promise<AffiliateNetwork>;
  updateNetwork: (id: string, data: UpdateNetworkInput) => Promise<AffiliateNetwork>;
  deleteNetwork: (id: string) => Promise<void>;

  // Link mutations
  createLink: (data: CreateLinkInput) => Promise<AffiliateLink>;
  updateLink: (id: string, data: UpdateLinkInput) => Promise<AffiliateLink>;
  deleteLink: (id: string) => Promise<void>;
}
```

Parallel fetch for networks, links, and stats. Re-export types from service.
