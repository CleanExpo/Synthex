# 51-02 Plan: Affiliate Link Manager UI

## Objective

Build dashboard UI for managing affiliate networks, links, and viewing analytics.

## Execution Context

- Follow patterns from Phase 50 (Sponsor CRM UI)
- Glassmorphic styling (bg-gray-900/50, border-white/10)
- PageHeader + DashboardEmptyState components
- Modal forms for CRUD operations
- Recharts for analytics visualization

## Context

### Components to Build
- NetworkList - configured affiliate networks
- NetworkForm - add/edit network
- LinkList - grid of affiliate links with stats
- LinkForm - add/edit link with auto-insertion settings
- LinkAnalytics - click/conversion charts
- Dashboard page with all components integrated

### Dependencies
- 51-01 backend complete
- useAffiliateLinks hook available
