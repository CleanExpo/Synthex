  <files>app/api/analytics/insights/route.ts</files>
  <action>
    The daily trends loop (lines ~191-216) executes up to 30 sequential prisma.post.count()
    calls — one per day. Replace with parallel Promise.all() to execute all counts in one
    concurrent batch.

    Pattern to replace:
    ```
    for (let i = 0; i < dayCount; i++) {
      const dayPosts = await prisma.post.count({...});  // sequential N+1
      trends.unshift({...});
    }
    ```

    Fix:
    1. Build dayRanges array first (no DB calls)
    2. Use Promise.all(dayRanges.map(...)) to run all counts concurrently
    3. Map results to trend objects, reverse for chronological order
  </action>
  <verify>npx tsc --noEmit 2>&1 | tail -5</verify>
  <done>N+1 eliminated. All 30 day counts run as parallel Promise.all. TypeScript clean.</done>
