# Phase 28 Plan 02: Collaboration Dashboard + Navigation

## Objective

Create useActivity hook and a unified Collaboration dashboard page with tabs for Activity Feed, Comments, and Shares. Add sidebar and command palette entries.

## Execution Context

**Existing infrastructure:**
- `app/api/activity/route.ts` - Activity API already exists (GET/POST)
- `components/realtime/LiveActivityFeed.tsx` - Real-time activity component exists
- WebSocketProvider with channel subscriptions

**New from Plan 28-01:**
- `hooks/use-comments.ts` - useComments hook
- `hooks/use-shares.ts` - useShares hook

**Pattern references:**
- Dashboard page: `app/dashboard/approvals/page.tsx`
- Sidebar: `app/dashboard/layout.tsx`
- Command palette: `components/CommandPalette.tsx`

## Tasks

### Task 1: Create useActivity Hook

**File:** `hooks/use-activity.ts`

**Read first:** `app/api/activity/route.ts` to understand response shape

**Exports:**
- `ActivityItem` type (matches API response)
- `ActivityFilter`: { userId?: string; contentType?: string; limit?: number }
- `useActivity(filter?: ActivityFilter)` hook

**Hook state:**
- `activities: ActivityItem[]`
- `loading: boolean`
- `error: string | null`

**Hook actions:**
- `refresh()`: Fetch activities with filter
- `loadMore()`: Pagination support

**Pattern:** Follow use-webhooks.ts (AbortController, useEffect fetch)

### Task 2: Create Collaboration Dashboard Page

**File:** `app/dashboard/collaboration/page.tsx`

**Page structure:**
- PageHeader with title "Team Collaboration"
- Tabs: Activity | Comments | Shares
- Tab content switches between views

**Activity Tab:**
- Use existing `LiveActivityFeed` component
- Or create simplified list using `useActivity` hook
- Show recent team activity with avatars, timestamps, action descriptions

**Comments Tab:**
- Filter: content type dropdown (all, campaigns, posts, projects)
- List of recent comments across all content
- Each comment shows: content preview, comment text, author, timestamp
- Click opens the content with comment highlighted
- Quick actions: resolve, reply

**Shares Tab:**
- Two sub-filters: "Shared with me" | "Shared by me"
- List of shared content items
- Each shows: content title, shared by/with, permission level, date
- Actions: revoke (for owner), open content

**Empty states:**
- Activity: "No recent activity. Start creating content to see team updates."
- Comments: "No comments yet. Start collaborating on content."
- Shares: "Nothing shared yet. Share content with team members."

**Styling:** Glassmorphic cards, consistent with approvals page

### Task 3: Add Sidebar and Command Palette Entries

**Sidebar (`app/dashboard/layout.tsx`):**
- Add entry: `{ icon: MessageSquare, label: 'Collaboration', href: '/dashboard/collaboration' }`
- Position: After Approvals, before Settings
- Import `MessageSquare` from icons barrel

**Command Palette (`components/CommandPalette.tsx`):**
