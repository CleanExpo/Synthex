## Tasks

### Task 1 — Prisma Models (3 new models + User back-relations)

Add to `prisma/schema.prisma`:

```prisma
model AwardListing {
  id              String    @id @default(cuid())
  userId          String
  orgId           String
  name            String
  organizer       String
  category        String
  deadline        DateTime?
  submissionUrl   String?
  status          String    @default("researched")
  description     String?   @db.Text
  nominationDraft String?   @db.Text
  entryFee        String?
  isRecurring     Boolean   @default(false)
  recurrenceNote  String?
  priority        String    @default("medium")
  notes           String?   @db.Text
  createdAt       DateTime  @default(now())
  updatedAt       DateTime  @updatedAt
  user            User      @relation(fields: [userId], references: [id])
  @@index([userId])
  @@index([deadline])
}

model DirectoryListing {
  id              String    @id @default(cuid())
  userId          String
  orgId           String
  directoryName   String
  directoryUrl    String
  listingUrl      String?
  category        String?
  status          String    @default("identified")
  domainAuthority Int?
  isFree          Boolean   @default(true)
  submittedAt     DateTime?
  lastReviewedAt  DateTime?
  notes           String?   @db.Text
  isAiIndexed     Boolean   @default(false)
  createdAt       DateTime  @default(now())
  updatedAt       DateTime  @updatedAt
  user            User      @relation(fields: [userId], references: [id])
  @@index([userId])
  @@index([status])
}

model SubmissionTracker {
  id              String    @id @default(cuid())
  userId          String
  orgId           String
  type            String
  referenceId     String
  action          String
  dueDate         DateTime?
  completedAt     DateTime?
  status          String    @default("pending")
  notes           String?
  createdAt       DateTime  @default(now())
  user            User      @relation(fields: [userId], references: [id])
  @@index([userId])
  @@index([dueDate])
}
```

Add User back-relations:
- `awardListings      AwardListing[]`
- `directoryListings  DirectoryListing[]`
- `submissionTrackers SubmissionTracker[]`

Run: `npx prisma db push`
