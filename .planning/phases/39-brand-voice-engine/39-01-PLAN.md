  <action>
Create usePersonas hook following the pattern from useApprovals/useWebhooks.

Include:
- `personas`: Persona[] state
- `isLoading`, `error` states
- `fetchPersonas()`: GET /api/personas
- `createPersona(data)`: POST /api/personas
- `updatePersona(id, data)`: PATCH /api/personas with id in body
- `deletePersona(id)`: DELETE /api/personas?id={id}
- `startTraining(id, sources)`: POST /api/personas/[id]/train
- `getTrainingStatus(id)`: GET /api/personas/[id]/train

Interface types:
```typescript
interface Persona {
  id: string;
  name: string;
  description: string | null;
  status: 'draft' | 'training' | 'active' | 'archived';
  tone: string;
  style: string;
  vocabulary: string;
  emotion: string;
  trainingSourcesCount: number;
  trainingWordsCount: number;
  trainingSamplesCount: number;
  accuracy: number;
  lastTrained: string | null;
  createdAt: string;
  updatedAt: string;
}

interface TrainingSource {
  type: 'text' | 'social_post' | 'document' | 'website' | 'conversation';
  content: string;
  metadata?: {
    platform?: string;
    engagement?: number;
    date?: string;
    url?: string;
  };
}
```

Use SWR pattern for caching (follow useApprovals pattern).
Include auth token from localStorage/sessionStorage.
