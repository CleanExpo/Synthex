## context

### Existing page structure

`app/dashboard/geo/page.tsx` has three tabs:
1. `analyze` — content textarea + submit
2. `results` — score overview + passages + recommendations (enabled only when `result !== null`)
3. `history` — placeholder

The `scoreDimensions` array drives the score bars on the Results tab. The `result` state is typed as `GEOAnalysisResult & { id: string }`.

### What Phase 85-01 added to the response

The API now returns `result.entityAnalysis: EntityAnalysisResult` containing:
- `entityCount: number`
- `uniqueEntityCount: number`
- `properNounDensity: number` (percentage)
- `entities: EntityMention[]` — each with `{ text, type, count, variants }`
- `coherenceIssues: string[]`
- `score: number` (0-100)

And `result.score.entityCoherence: number` is available.

### UI design

**Entity tab content** (added inside the Results TabsContent):

1. **Entity Coherence Score card** (alongside the overall score):
   - Large number: `result.score.entityCoherence`
   - Label: "Entity Coherence"
   - Sub-label: `result.entityAnalysis.entityCount` entities found, `result.entityAnalysis.properNounDensity.toFixed(1)`% density
   - Tier badge (same `getTier()` function)
   - Target indicator: "Target: 15+ entities, ~20.6% density"

2. **Entity type breakdown** — 4 pills showing count per type:
   - PERSON (blue), ORGANISATION (purple), LOCATION (green), CONCEPT (amber)
   - Format: "👤 3 Persons", "🏢 5 Orgs", etc.

3. **Entity list** — top 20 entities, sorted by count:
   - Each row: type badge + entity name + occurrence count + variant chips
   - Color-coded badges: PERSON=blue, ORGANISATION=purple, LOCATION=emerald, CONCEPT=amber

4. **Coherence issues** — if any:
   - Warning card listing coherence issues from `result.entityAnalysis.coherenceIssues`
   - If none: success message "No coherence issues detected"

### Implementation approach

Add a fourth tab trigger "Entities" to `TabsList`. Add the corresponding `TabsContent value="entities"`. The tab is disabled when `!result` (same as "results" tab).

Add `entityCoherence` to the `scoreDimensions` array with a `Boxes` or `Network` icon (from existing icon imports — check what's available; use `Target` or `BarChart3` if nothing better).

The `scoreDimensions` array and score bars in results currently only show 5 dimensions — after 85-01, `entityCoherence` is on `GEOScore` and can be added as a 6th bar with weight "bonus" or "(diagnostic)".
