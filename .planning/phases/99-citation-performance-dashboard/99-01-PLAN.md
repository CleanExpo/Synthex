# Phase 99-01 — Citation Performance Dashboard: Aggregation API Layer

## Objective
Build a read-only aggregation API layer that pulls data from all v5.0 Phase tables (85–98) into three unified endpoints used by the citation command centre dashboard.

## Prerequisites
- All Phase 85–98 models confirmed in `prisma/schema.prisma` (no new models needed)
- `APISecurityChecker` + `DEFAULT_POLICIES.AUTHENTICATED_READ` available
- `prisma` client imported from `lib/prisma`
- `logger` available from `lib/logger`

## No Prisma Changes
This phase is purely read-only aggregation. Zero schema modifications.

## Service File

### lib/citation/aggregator.ts
Three exported async functions:

**`getOverviewStats(userId, orgId)`** — aggregates:
- `GEOAnalysis`: avg overallScore, count, latest createdAt
- `AuthorityAnalysis`: avg overallScore, count
- `ContentQualityAudit`: avg humanessScore, count, pass rate
- `EEATAudit`: avg overallScore, count
- `BrandIdentity`: count, avg consistencyScore, kgmid presence (knowledgePanelFound)
- `BrandMention`: count of mentions in last 24h
- `CitationMonitor`: total monitored, sum citationCount, cited vs not-cited
- `SentinelAlert`: unacknowledged count, critical count, last 7 days count
- `PromptTracker`: total tracked, total mentioned (brandMentioned=true), coverage rate
- `BacklinkProspect`: counts by status (identified/contacted/published)
- `SEOExperiment`: running count, completed count, avg improvement
- `HealingAction`: recent count (last 7 days)

Agent activity derived from:
- Last 2 `SentinelAlert` rows → "Sentinel" agent
- Last 2 `HealingAction` rows → "Self-Healing" agent
- Last 2 `ContentQualityAudit` rows → "Quality Gate" agent
- Last 2 `PromptTracker` rows (where lastTestedAt not null) → "Prompt Intelligence" agent
- Last 2 `BacklinkProspect` rows → "Backlink" agent

All wrapped in try-catch; failures return zeros, not errors.

**`getTimeline(userId, orgId, days)`** — returns array of `{ date, geoScore, qualityScore, alertCount }`:
- Groups `GEOAnalysis.createdAt` and `ContentQualityAudit.createdAt` by day
- Groups `SentinelAlert.createdAt` by day for alert counts
- Merges into unified day-keyed map, fills missing days with nulls
- Returns last N days sorted ascending

**`getOpportunities(userId, orgId)`** — returns up to 10 priority items:
- Unacknowledged `SentinelAlert` with severity='critical' → type='alert', severity='critical'
- Unacknowledged `SentinelAlert` with severity='warning' → type='alert', severity='warning'
- `PromptTracker` where `lastTestedAt` is null → type='prompt', severity='warning'
- `BacklinkProspect` where status='identified' → type='backlink', severity='info'
- `SEOExperiment` where status='running' → type='experiment', severity='info'
Each has: `{ type, title, description, severity, href }`

## API Routes

### GET /api/citation/overview
File: `app/api/citation/overview/route.ts`
- Auth via `APISecurityChecker.check(request, DEFAULT_POLICIES.AUTHENTICATED_READ)`
- Calls `getOverviewStats(userId, orgId ?? userId)`
- Returns `{ success: true, data: OverviewStats }`
- Try-catch → 500 on failure

### GET /api/citation/timeline
File: `app/api/citation/timeline/route.ts`
- Auth same as above
- Query param: `days` (default 30, max 90)
- Calls `getTimeline(userId, orgId ?? userId, days)`
- Returns `{ success: true, data: TimelinePoint[] }`

### GET /api/citation/opportunities
File: `app/api/citation/opportunities/route.ts`
- Auth same as above
- Calls `getOpportunities(userId, orgId ?? userId)`
- Returns `{ success: true, data: OpportunityItem[] }`

## TypeScript Types (inline in aggregator.ts)
```typescript
interface AgentActivity {
  agent: 'sentinel' | 'healing' | 'quality' | 'prompt' | 'backlink' | 'experiment'
  action: string
  createdAt: Date
  href: string
}
interface OverviewStats {
  geoSummary: { avgScore: number; count: number; lastAnalyzedAt: Date | null }
  citationSummary: { totalMonitored: number; cited: number; notCited: number; newMentions24h: number }
  entitySummary: { profilesCreated: number; avgCoherenceScore: number; knowledgePanelFound: number }
  qualitySummary: { avgQualityScore: number; avgEeatScore: number; auditsRun: number }
  promptSummary: { tracked: number; mentioned: number; coverageRate: number }
  backlinkSummary: { prospectsFound: number; contacted: number; published: number }
  alertsSummary: { unacknowledged: number; critical: number; recent7Days: number }
  experimentsSummary: { running: number; completed: number; avgImprovement: number }
  agentActivity: AgentActivity[]
}
interface TimelinePoint {
  date: string // YYYY-MM-DD
  geoScore: number | null
  qualityScore: number | null
  alertCount: number
}
interface OpportunityItem {
  type: 'alert' | 'prompt' | 'backlink' | 'experiment'
  title: string
  description: string
  severity: 'critical' | 'warning' | 'info'
  href: string
}
```
