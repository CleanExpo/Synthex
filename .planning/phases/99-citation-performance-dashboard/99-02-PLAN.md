# Phase 99-02 — Citation Performance Dashboard: Unified Command Centre UI

## Objective
Build the unified `/dashboard/citation` command centre page that aggregates data from all v5.0 AI-Native GEO & Citation Engine phases into a single high-level read-only view.

- Phase 99-01 API routes live at `/api/citation/overview`, `/api/citation/timeline`, `/api/citation/opportunities`
- SWR installed (already present — see `GamificationWidget.tsx`)

- `CommandLineIcon` exported from `@/components/icons`

## Components

### components/citation/CitationScoreGauge.tsx
**Props:** `{ geoScore, qualityScore, eeatScore, authorityScore, promptCoverage }`
- Computes overall: `geo×0.25 + quality×0.20 + eeat×0.20 + authority×0.15 + prompt×0.20`
- Displays large number (0-100) with grade letter: A (≥90), B (≥80), C (≥60), D (≥40), F (<40)
- Colour ring: emerald (≥80), cyan (≥60), amber (≥40), red (<40)
- SVG circle gauge (stroke-dasharray trick, no library)
- Sub-scores as small labelled pills below

### components/citation/AgentActivityFeed.tsx
**Props:** `{ activities: AgentActivity[], loading: boolean }`
- List of up to 10 items
- Agent icon by type: shield (sentinel), sparkles (healing/quality), beaker (experiment), link (backlink), brain (prompt)
- Agent colour by type: cyan (sentinel), emerald (healing), amber (quality), purple (prompt), blue (backlink)
- Each item: icon + agent label + action text + relative time ("2 hours ago")
- Link to relevant dashboard page
- Loading: 3 skeleton rows
- Empty: "No recent agent activity"

### components/citation/OpportunityPipeline.tsx
**Props:** `{ items: OpportunityItem[], loading: boolean }`
- Priority sorted: critical first, then warning, then info
- Each item: severity chip + description + "Go →" link
- Chip colours: critical=red, warning=amber, info=blue
- Loading: 3 skeleton rows
- Empty: "All clear — no action items" with green checkmark

### components/citation/ScoreTrendChart.tsx
**Props:** `{ data: TimelinePoint[], loading: boolean }`
- Pure SVG, no charting library
- Viewbox 800×200, responsive via `width="100%"`
- Three lines: GEO (cyan), Quality (emerald), Alerts normalised (amber)
- X-axis: last 30 days, weekly labels (every 7th tick)
- Y-axis: 0–100, 4 guide lines at 25/50/75/100
- Legend: coloured squares + labels
- Null values create gaps (polyline split at nulls)
- Loading: shimmer rectangle placeholder

### components/citation/ModuleScoreGrid.tsx
**Props:** `{ overview: OverviewStats, loading: boolean }`
- 11 module cards in responsive CSS grid (2 cols mobile, 3 cols tablet, 4 cols desktop)
- Modules with score source:
  1. GEO Analysis → `geoSummary.avgScore`
  2. Entity Coherence → (derived from GEO entity score, shown as `—` if no data)
  3. Authority Engine → `entitySummary.avgCoherenceScore`
  4. Writing Quality → `qualitySummary.avgQualityScore`
  5. AI Slop Detection → `qualitySummary.avgQualityScore` (humaness proxy)
  6. E-E-A-T Builder → `qualitySummary.avgEeatScore`
  7. Brand Builder → `entitySummary.profilesCreated` count (not a score — shows "N profiles")
  8. PR Manager → `backlinkSummary.contacted` count
  9. Backlink Prospector → `backlinkSummary.prospectsFound` count
  10. Prompt Intelligence → `promptSummary.coverageRate`%
  11. Algorithm Sentinel → `alertsSummary.recent7Days` alerts (info badge)
- Status dot: green pulse if count/score > 0, grey if 0
- Link to relevant `/dashboard/` page
- Loading: grid of skeleton cards

## Page

### app/dashboard/citation/page.tsx
`'use client'` — SWR-driven with 60s refresh interval.

Three SWR hooks:
```typescript
const { data: overview } = useSWR('/api/citation/overview', fetchJson, { refreshInterval: 60000 })
const { data: timeline } = useSWR('/api/citation/timeline?days=30', fetchJson, { refreshInterval: 60000 })
const { data: opportunities } = useSWR('/api/citation/opportunities', fetchJson, { refreshInterval: 60000 })
```

Layout (single page, no tabs):

Tailwind classes: `grid grid-cols-1 lg:grid-cols-3` for top row (gauge 1 col, chart 2 cols).
