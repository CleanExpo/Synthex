- GeoScoreTrend: { date: string, overall: number, citability: number, structure: number, multiModal: number, authority: number, technical: number }

Hook state:
- loading: boolean (for analyze)
- historyLoading: boolean
- trendsLoading: boolean
- error: string | null
- result: GeoReadinessResult | null

- loadTrends(days?: number) — GET /api/seo/geo-readiness/trends?days=N, sets trends

Use useCallback for all functions. Include abort controller pattern for cleanup. Use credentials: 'include' on all fetch calls.

Auto-load history and trends on mount (useEffect with loadHistory() and loadTrends()).
