<task id="2" name="Create lib/voice/types.ts">
Create the file with ALL type definitions for the voice module.

```typescript
/**
 * Voice Fingerprinting & Content Capsule Types
 * @module lib/voice/types
 */

export interface SentenceStats {
  mean: number;    // avg words per sentence
  stdDev: number;  // variation (higher = more varied rhythm)
  min: number;
  max: number;
}

export interface VoiceFingerprint {
  // Sentence rhythm
  sentenceLengths: SentenceStats;
  // Vocabulary
  ttr: number;             // Type-Token Ratio 0.0-1.0 (higher = more diverse)
  avgWordLength: number;   // chars (higher = more sophisticated vocabulary)
  // Style markers
  punctuationDensity: number;    // non-letter chars per sentence
  emDashRate: number;            // em dashes per 100 words
  questionRate: number;          // ? per 100 words
  firstPersonRate: number;       // I/me/my per total words
  adverbDensity: number;         // -ly words per 100 words
  passiveVoiceEstimate: number;  // 0.0-1.0 (higher = more passive)
  // Readability
  fleschReadingEase: number;     // 0-100 (higher = easier to read)
  avgParagraphLength: number;    // words per paragraph
  // Sample metadata
  sampleWordCount: number;
  sampleSentenceCount: number;
  extractedAt: string;  // ISO timestamp
}

export interface FingerprintResult {
  valid: boolean;
  fingerprint?: VoiceFingerprint;
  error?: string;  // e.g. 'Insufficient sample — minimum 200 words required'
}

// Content Capsule types
export interface CapsuleSection {
  heading: string;        // Question-format H2/H3
  answerBlock: string;    // 75-120 word direct answer
  supporting: string[];   // Bullet points of evidence
  wordCount: number;
  isCapsuleReady: boolean;  // Already meets the 75-120 word spec
}

export interface ContentCapsuleResult {
  originalWordCount: number;
  capsuleWordCount: number;
  sections: CapsuleSection[];
  extractabilityScore: number;  // 0-100, higher = more AI-extractable
  isAlreadyCapsuleReady: boolean;
}

// Slop scanner types
export type SlopCategory = 'transition' | 'filler' | 'overused-word' | 'structural-pattern' | 'hedge';
export type SlopSeverity = 'error' | 'warning';

export interface SlopMatch {
  phrase: string;
  category: SlopCategory;
  severity: SlopSeverity;
  startIndex: number;
  endIndex: number;
  suggestion?: string;  // Optional better alternative
}

export interface SlopScanResult {
  totalMatches: number;
  errorCount: number;
  warningCount: number;
  slopDensity: number;  // matches per 100 words
  matches: SlopMatch[];
  overallSeverity: SlopSeverity | 'clean';
  wordCount: number;
}

// Writing context types
export interface WritingContextResult {
  systemPrompt: string;   // Ready-to-inject system prompt text
  tokenEstimate: number;  // Approximate token count
  compact: boolean;       // Whether compact mode was used
}

// Orchestrator result
export interface VoiceAnalysisResult {
  fingerprint: FingerprintResult;
  slopScan: SlopScanResult;
  writingContext?: WritingContextResult;
}
```

<task id="3" name="Create lib/voice/fingerprint-extractor.ts">
Pure TypeScript stylometric analysis. Extract all metrics defined in VoiceFingerprint.

Key implementation details:
- Split sentences on /[.!?]+(?=\s|$)/ — handles abbreviations better than naive split
- TTR: uniqueWords/totalWords using Set
- Flesch Reading Ease: 206.835 - 1.015*(words/sentences) - 84.6*(syllables/words)
- Syllable approximation: count vowel groups [aeiouy]+, subtract trailing silent-e
- Em dash rate: count both — and ` - ` (spaced hyphen)
- First person rate: /\b(I|me|my|mine|myself)\b/gi count / total words
- Adverb density: /\b\w+ly\b/gi count per 100 words
- Passive voice estimate: sentences matching /\b(was|were|is|are|been)\b.*\bby\b/i / total sentences
- Paragraph length: split on /\n\n+/, count words per paragraph, average
- Return { valid: false, error: '...' } if wordCount < 200
- Round all floats to 2 decimal places for clean display
</task>

- "nuanced" — warning
- "multifaceted" — warning

- "paramount" — error

STRUCTURAL_PATTERN (AI templates):
- "not only X but also Y" → regex /not only .{3,50} but also/i — error
