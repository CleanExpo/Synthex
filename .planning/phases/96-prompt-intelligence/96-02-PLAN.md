## Objective

Build the Prompt Intelligence dashboard at `/dashboard/prompts` with 3 tabs (Discovery, Tracking, Gaps), 4 reusable components, sidebar entry in the PR MANAGER group, and 2 command palette entries.

## Components

### components/prompts/PromptCard.tsx
Card for a single tracked prompt:
- Prompt text (truncated to 120 chars) with full text on hover
- Category badge with colour coding:
  - brand-awareness: purple
  - competitor-comparison: amber
  - local-discovery: green
  - use-case: blue
  - how-to: cyan
  - product-feature: slate
- Status indicator: Not tested (grey) | ✓ Brand mentioned (green) | ✗ Not mentioned (red)
- Position chip if mentioned: "#1 mention", "#2 mention", etc.
- "Test" button → POST /api/prompts/test
- Loading spinner during test

### components/prompts/PromptGapChart.tsx
Visual gap analysis panel:
- Mention rate donut/ring using CSS (mentioned vs not-mentioned)
- Category breakdown: horizontal progress bars showing mention rate per category
- Gap list: categories with lowest mention rate + recommended actions

### components/prompts/CompetitorVisibilityTable.tsx
Table of competitors found in AI responses:
| Competitor | Mentions | Mention Rate | Avg Position |
- Populated from PromptResult.competitorsFound aggregation
- Sorted by mention count descending
- Empty state when no competitors detected

### components/prompts/PromptGeneratorForm.tsx
Form to generate prompt templates:
- Fields: Entity Name (text), Entity Type (select: brand/product/service/person/location), Topic/Industry (text), Location (optional text)
- "Generate Prompts" button → POST /api/prompts/generate
- Displays generated prompts list
- Each prompt: category badge + "Add to Tracker" button (POST /api/prompts/trackers)
- Bulk "Track All" button

## Page: app/dashboard/prompts/page.tsx

3-tab layout:

### Tab 1: Discovery
- PromptGeneratorForm at top
- Generated prompts list below (filtered by category)
- Bulk "Track All" button + individual "Track" buttons per prompt

### Tab 2: Tracking
- SWR data fetch from /api/prompts/trackers
- Filter bar: by status (pending/tested) and category
- List of PromptCards
- "Test All Untested" bulk action (max 5 concurrent, sequential)
- Summary stat row: total tracked, tested, mentioned, coverage %

### Tab 3: Gaps
- SWR data fetch from /api/prompts/gaps
- PromptGapChart (donut + category bars)
- CompetitorVisibilityTable
- Recommendations text block
