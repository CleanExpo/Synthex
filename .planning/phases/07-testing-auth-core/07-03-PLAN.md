<task type="auto">
  <name>Task 2: Create representative platform service tests</name>
  <files>tests/unit/lib/social/platform-services.test.ts</files>
  <action>
Create tests for 3 representative platform services that cover different API integration patterns:
- **TwitterSyncService** — OAuth Bearer token, JSON API
- **InstagramService** — Meta Graph API, container-based posting
- **RedditService** — Form-encoded POST, User-Agent required, public endpoints

**Setup:**
- Mock global fetch with jest.fn()
- Create test credentials: { accessToken: 'test-token', refreshToken: 'test-refresh', platformUserId: 'test-user-123', expiresAt: futureDate }
- Mock onTokenRefresh callback with jest.fn()

**Test categories per service:**

### Twitter (OAuth Bearer pattern):
1. **syncProfile()** — Mock fetch returning user profile JSON → verify correct mapping to SyncProfileResult
2. **createPost()** — Mock fetch returning tweet creation response → verify correct request body and result
3. **Token refresh** — Mock 401 response then success → verify refresh callback called and retry succeeds
4. **Error handling** — Mock network error → verify graceful failure (no throw)

### Instagram (Meta Graph API pattern):
1. **syncProfile()** — Mock Graph API response → verify mapping
2. **createPost()** — Verify two-step flow: POST to create container, then POST to publish
3. **syncPosts()** — Mock paginated response → verify pagination cursor handling
4. **Error handling** — Mock Graph API error response → verify parsed correctly

### Reddit (Form-encoded + public pattern):
1. **syncProfile()** — Mock /api/v1/me response → verify mapping
2. **createPost()** — Verify URLSearchParams body (not JSON), correct Content-Type header
3. **User-Agent header** — Verify all requests include User-Agent: Synthex/1.0
4. **deletePost()** — Verify t3_ prefix added to post ID
5. **Error handling** — Mock rate limit (429) → verify graceful failure

### Cross-service patterns:
1. **All services implement same interface** — syncProfile, syncPosts, syncAnalytics, createPost, deletePost, getPostMetrics all exist
2. **Token expiry detection** — isTokenExpired() returns true when past expiresAt
3. **Credentials management** — isConfigured() returns true with valid tokens, false without

**Mocking strategy:**
- Use fetch mock that routes responses by URL pattern:
  ```
  if (url.includes('api.twitter.com')) return twitterResponse;
  if (url.includes('graph.facebook.com')) return instagramResponse;
  if (url.includes('oauth.reddit.com')) return redditResponse;
  ```
- Reset mocks between test suites (beforeEach)

**Test count target:** 20-25 test cases across 3 services + cross-service patterns.

Don't: Test all 9 services individually — 3 representative services prove the pattern works.
Don't: Test actual API responses — mock all fetch calls.
Don't: Test factory creation — that's covered in Task 1.
