  <name>Task 1: Create MentionFetcher service</name>
  <files>lib/social/mention-fetcher.ts</files>
  <action>
Create service following competitor-fetcher.ts pattern:

```typescript
export interface FetchedMention {
  platform: string;
  platformPostId: string;
  platformUrl: string;
  authorHandle: string;
  authorName: string | null;
  authorAvatar: string | null;
  authorFollowers: number | null;
  content: string;
  mediaUrls: string[];
  likes: number;
  comments: number;
  shares: number;
  postedAt: Date;
  success: boolean;
  error?: string;
}

export async function fetchMentions(
  keyword: string,
  platform: string,
  accessToken: string | null,
  since?: Date
): Promise<FetchedMention[]>
```

Platform implementations:
1. **Twitter/X**: Search API v2 with query containing keyword
   - Requires Bearer token
   - Search recent tweets (7-day window)
   - Include user expansions for author info

2. **Instagram**: Limited - Business Discovery only works for specific accounts
   - Return empty with error message noting limitation

3. **YouTube**: Comments search API
   - Search for comments containing keyword on user's videos
   - Requires API key

4. **Reddit**: Public search endpoint
   - No auth needed
   - Search posts and comments

Design: Never throws, returns array with success flags per mention
  </action>
  <verify>npm run type-check passes</verify>
  <done>MentionFetcher fetches mentions from Twitter, YouTube, Reddit</done>
