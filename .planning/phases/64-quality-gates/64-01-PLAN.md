# Phase 64-01 Plan: AI Quality & Brand Voice Guardian

## Goal
Auto-review AI-generated content before publishing. Confidence scoring routes
low-confidence content to human review. Integrates with Phase 62 WorkflowExecution.

## Architecture
- QualityScorer calls OpenRouter (claude-3-haiku) with brand voice profile as system context
- Persona model already exists — use tone/style/vocabulary/emotion fields as BrandVoiceProfile
- High confidence (≥ 0.85) → auto-approved; Low confidence → waiting_approval
- Review queue = StepExecution records with status 'waiting_approval'
- human-approval.ts step type updated to optionally call QualityScorer
