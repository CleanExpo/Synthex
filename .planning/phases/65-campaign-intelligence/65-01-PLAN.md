# Phase 65-01 Plan: Campaign Intelligence Engine

## Goal
Performance-based learning. StepExecution outputs + approval decisions feed back
to improve prompt templates. Context > model — improve context assembly, not model version.

## Architecture
- Analyse historical StepExecution records (last 50 completed for a template)
- Group by stepIndex; calculate avg confidenceScore per prompt variation
- Use OpenRouter to suggest improved prompts for low-confidence steps
- NEVER auto-applies — suggestions returned for human approval
- `WorkflowTemplate.steps[].promptTemplate` updated only on human approval
