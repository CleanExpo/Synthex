<task type="auto">
  <name>Task 1: Create report builder hooks</name>
  <files>hooks/use-report-templates.ts, hooks/use-report-export.ts</files>
  <action>
Create two hooks:

**useReportTemplates:**
- Fetch templates from GET /api/reports/templates (with optional category filter)
- Save template via POST /api/reports/templates (accepts name, description, category, reportType, metrics, dimensions, visualizations, layout)
- Update template via PATCH /api/reports/templates?id=X
- Delete template via DELETE /api/reports/templates?id=X
- Return { templates, isLoading, error, saveTemplate, updateTemplate, deleteTemplate, refetch }
- Follow the pattern from use-performance-analytics.ts (useState + useCallback + fetch)

**useReportExport:**
- Generate report via POST /api/reporting/generate (accepts name, type, format, dateRange)
- Poll status via GET /api/reporting/reports/[reportId] every 2s until completed/failed (max 30 attempts)
- Return { generateReport, isGenerating, exportStatus }
- On completion, return the download URL from the report data

Both hooks must:
- Use credentials: 'include' on all fetch calls
- Handle errors gracefully with try/catch
- Return typed data (define interfaces in each hook file)
- NOT use SWR/TanStack Query (follow existing pattern of raw fetch + useState)
