# Phase 66-01 Plan: Autonomous Insights Agent

## Goal
Scheduled AI proactively surfaces content opportunities and auto-drafts top content.
Triggered by Vercel cron, bounded by circuit breakers.

## Architecture
- Vercel cron triggers `/api/cron/insights` every 4 hours
- Circuit breaker: check last run timestamp in DB via WorkflowExecution records
- Agent analyses PlatformMetrics (last 7 days) + SocialMention (recent) for opportunities
- Generates top 3 content opportunities + draft content for each
- High-confidence (≥ 0.85) → auto-draft as Campaign+Post (pending, not published)
- Low-confidence → WorkflowExecution with status waiting_approval
- CRON_SECRET header validates cron requests
