# Plan 79-03: Audit Log Detail Drawer

## objective

Add a slide-out drawer to the Audit Log tab that shows full details of an audit log entry when a row is clicked. Currently, the audit log table shows truncated data in columns (timestamp, user email, action, resource, severity, outcome) but the `details` JSON field -- which contains the richest information (reason, target email, changed fields, IP addresses, error messages) -- is completely hidden. Users must query the database directly to see this information.

Linear: SYN-18
Phase dir: `.planning/phases/79-admin-completion/`

- Audit log viewer: `components/admin/audit-log-viewer.tsx` -- table with pagination and filters, no row click handler
- Audit log API: `app/api/admin/audit-log/route.ts` -- returns entries with `details` JSON, `ipAddress`, `userAgent` fields

## context

### Discovery Findings

**Gap**: The audit log API returns `details`, `ipAddress`, and `userAgent` fields in each entry, but the `AuditLogViewer` component does not display them. The `AuditLogEntry` interface includes `details?: Record<string, unknown> | null` but it is never rendered. There is no row click handler and no detail view.

**Use cases for detail view:**
1. Admin suspends a user -- details contains `{ reason, targetEmail }` -- admin needs to see why
2. Login failure -- details may contain `{ error, attemptedEmail, ipAddress }` -- security audit
3. Subscription change -- details contains `{ plan, limits, targetUserId }` -- billing investigation
4. API key creation/revocation -- details contains key metadata

### Architecture Decision

- Use a Sheet (slide-out drawer from the right) rather than a modal dialog -- it preserves the table context and allows comparison
- The drawer receives the full `AuditLogEntry` and renders all fields in a structured layout
- No additional API call needed -- the list endpoint already returns complete entries including `details`
- Add `ipAddress` and `userAgent` to the `AuditLogEntry` interface (already in API response, just not typed)

### Task 2: Create `components/admin/audit-log-drawer.tsx`

New component that renders the full detail of an audit log entry.

**Props:**
```ts
interface AuditLogDrawerProps {
  entry: AuditLogEntry | null;
  open: boolean;
  onClose: () => void;
}
```

**Layout sections:**

1. **Header**: Action name as title, severity badge, outcome badge, close button
2. **Metadata grid**: Two-column grid showing:
   - Timestamp (formatted with timezone)
   - User (email or ID)
   - Resource type + resource ID (linked if possible)
   - Category
   - Severity
   - Outcome
3. **Details section**: Render the `details` JSON as key-value pairs:
   - For string values: show directly
   - For object/array values: render as formatted JSON in a code block
   - For `targetEmail`: highlight as a linked value
   - For `reason`: show in a callout/alert box
   - For `changes`: show as a before/after comparison
4. **Request info** (collapsible): IP address, user agent (parsed to show browser/OS if possible)
5. **Resource ID** with copy button

**Styling**: Glassmorphic dark theme consistent with the rest of the admin panel.

```tsx
'use client';

import { Sheet, SheetContent, SheetHeader, SheetTitle } from '@/components/ui/sheet';
// OR Dialog-based implementation if Sheet is unavailable
import { Badge } from '@/components/ui/badge';
import { Button } from '@/components/ui/button';
import { Copy, X } from '@/components/icons';
import { toast } from 'sonner';

const SEVERITY_COLOURS: Record<string, string> = {
  low: 'bg-gray-500/20 text-gray-300',
  medium: 'bg-yellow-500/20 text-yellow-400',
  high: 'bg-orange-500/20 text-orange-400',
  critical: 'bg-red-500/20 text-red-400',
};

const OUTCOME_COLOURS: Record<string, string> = {
  success: 'bg-green-500/20 text-green-400',
  failure: 'bg-red-500/20 text-red-400',
};

export function AuditLogDrawer({ entry, open, onClose }: AuditLogDrawerProps) {
  if (!entry) return null;

  const copyToClipboard = (text: string) => {
    navigator.clipboard.writeText(text);
    toast.success('Copied to clipboard');
  };

  const details = entry.details as Record<string, unknown> | null;

  return (
    <Sheet open={open} onOpenChange={(o) => !o && onClose()}>
      <SheetContent className="w-[480px] bg-gray-950 border-white/10 overflow-y-auto">
        <SheetHeader>
          <SheetTitle className="text-white font-mono text-sm">
            {entry.action}
          </SheetTitle>
        </SheetHeader>

        {/* Badges */}
        <div className="flex gap-2 mt-2">
          <Badge className={SEVERITY_COLOURS[entry.severity]}>
            {entry.severity}
          </Badge>
          <Badge className={OUTCOME_COLOURS[entry.outcome]}>
            {entry.outcome}
          </Badge>
          <Badge className="bg-white/10 text-gray-300">
            {entry.category}
          </Badge>
        </div>

        {/* Metadata Grid */}
        <div className="grid grid-cols-2 gap-3 mt-6">
          <MetadataItem label="Timestamp" value={new Date(entry.createdAt).toLocaleString()} />
          <MetadataItem label="User" value={entry.user?.email ?? entry.userId} />
          <MetadataItem label="Resource" value={entry.resource} />
          <MetadataItem label="Resource ID" value={entry.resourceId ?? 'N/A'} copyable />
        </div>

        {/* Details Section */}
        {details && Object.keys(details).length > 0 && (
          <div className="mt-6">
            <h4 className="text-sm font-medium text-gray-300 mb-3">Details</h4>
            <div className="space-y-2">
              {Object.entries(details).map(([key, value]) => (
                <DetailRow key={key} label={key} value={value} />
              ))}
            </div>
          </div>
        )}

        {/* Request Info */}
        {(entry.ipAddress || entry.userAgent) && (
          <div className="mt-6">
            <h4 className="text-sm font-medium text-gray-300 mb-3">Request Info</h4>
            <div className="space-y-2 text-xs">
              {entry.ipAddress && (
                <div className="flex justify-between items-center p-2 bg-white/5 rounded">
                  <span className="text-gray-400">IP Address</span>
                  <span className="text-white font-mono">{entry.ipAddress}</span>
                </div>
              )}
              {entry.userAgent && (
                <div className="p-2 bg-white/5 rounded">
                  <span className="text-gray-400">User Agent</span>
                  <p className="text-white font-mono text-xs mt-1 break-all">
                    {entry.userAgent}
                  </p>
                </div>
              )}
            </div>
          </div>
        )}

        {/* Entry ID */}
        <div className="mt-6 pt-4 border-t border-white/10">
          <div className="flex items-center justify-between">
            <span className="text-xs text-gray-500">Entry ID</span>
            <Button
              size="sm"
              variant="ghost"
              onClick={() => copyToClipboard(entry.id)}
              className="text-gray-500 hover:text-white text-xs"
            >
              {entry.id.slice(0, 12)}...
              <Copy className="w-3 h-3 ml-1" />
            </Button>
          </div>
        </div>
      </SheetContent>
    </Sheet>
  );
}
```

Create `MetadataItem` and `DetailRow` as internal helper components within the file.
