**Established patterns:**
- `useApi` hook in hooks/use-api.ts — SWR-inspired with caching, polling, revalidation
- `use-dashboard.ts` — wraps useApi with preset staleTime/cacheTime per endpoint
- Recharts dark theme: `contentStyle={{ backgroundColor: 'rgba(0,0,0,0.8)', border: '1px solid #333', borderRadius: '8px' }}`, CartesianGrid `strokeDasharray="3 3" stroke="#333"`, axis `stroke="#666"`
- Glassmorphic cards: `variant="glass"`, bg-white/5, border-white/10

**Constraining decisions:**
- [Phase 16]: Loading states use glassmorphic styling
- [Phase 16]: DashboardError component for all error.tsx files
- [Phase 20/21]: Inline fetch pattern acceptable for simple pages, but analytics page complexity warrants hook

**Performance API response shape (already implemented):**
```typescript
{
  overview: { totalPosts, totalEngagement, averageEngagementRate, totalReach, totalImpressions },
  growth: { engagementChange, reachChange, postsChange, trend: 'up'|'down'|'stable' },
  timeline: Array<{ date, engagement, reach, impressions, posts }>,
  platforms: Array<{ platform, engagement, engagementRate, posts, bestTime }>,
  topContent: Array<{ id, content, platform, engagement, engagementRate, publishedAt }>
}
```
