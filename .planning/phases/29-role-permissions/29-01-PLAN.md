# Phase 29 Plan 01: Roles API + useRoles Hook

## Objective

Create CRUD API routes for roles management using existing RoleManager service, plus React hook for state management.

## Execution Context

**Existing infrastructure:**
- `lib/auth/rbac/role-manager.ts` - RoleManager with createRole, updateRole, deleteRole, grantRole, revokeRole, getRoles, getUserRoles
- `lib/auth/rbac/permission-engine.ts` - PermissionEngine with check, getUserPermissions, ALL_PERMISSIONS
- Prisma models: Role, UserRole, PermissionAudit

**Hook pattern:** Follow `hooks/use-webhooks.ts` pattern (raw fetch + useState, AbortController)

**API pattern:** Follow `app/api/webhooks/user/route.ts` pattern (auth, validation, error handling)

## Tasks

### Task 1: Create Roles CRUD API

**Files to create:**
- `app/api/roles/route.ts` - GET (list org roles), POST (create role)
- `app/api/roles/[id]/route.ts` - GET, PATCH, DELETE
- `app/api/roles/[id]/users/route.ts` - GET (users with role), POST (grant), DELETE (revoke)

**GET /api/roles:**
- Requires auth session
- Gets organizationId from session
- Returns all roles for org using RoleManager.getRoles()
- Also returns ALL_PERMISSIONS for UI dropdown

**POST /api/roles body:**
- `name`, `permissions[]` (required)
- `description`, `isDefault` (optional)
- Uses RoleManager.createRole()

**GET /api/roles/[id]:**
- Returns single role with user count

**PATCH /api/roles/[id] body:**
- `name`, `description`, `permissions`, `isDefault` (all optional)
- Uses RoleManager.updateRole()

**DELETE /api/roles/[id]:**
- Uses RoleManager.deleteRole()
- Fails if role has assigned users

**GET /api/roles/[id]/users:**
- Returns users with this role using RoleManager.getUsersWithRole()

**POST /api/roles/[id]/users body:**
- `userId` (required), `expiresAt` (optional)
- Uses RoleManager.grantRole()

**DELETE /api/roles/[id]/users?userId=xxx:**
- Uses RoleManager.revokeRole()

**Auth:** Require session, require roles:manage permission
**Validation:** Zod schemas for request bodies

### Task 2: Create useRoles Hook

**File:** `hooks/use-roles.ts`

**Exports:**
- `Role` type (matches API response)
- `Permission` type (from permission-engine)
- `CreateRoleData`, `UpdateRoleData` types
- `useRoles()` hook

**Hook state:**
- `roles: Role[]`
- `availablePermissions: string[]`
- `loading: boolean`
- `error: string | null`

**Hook actions:**
- `refresh()`: Fetch roles for current org
- `create(data: CreateRoleData): Promise<Role>`
- `update(id: string, data: UpdateRoleData): Promise<void>`
- `remove(id: string): Promise<void>`
- `grantToUser(roleId: string, userId: string, expiresAt?: Date): Promise<void>`
- `revokeFromUser(roleId: string, userId: string): Promise<void>`
- `getUsersWithRole(roleId: string): Promise<User[]>`

**Pattern:** Follow use-webhooks.ts (AbortController, useEffect fetch, callbacks)
