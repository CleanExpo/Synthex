<task type="auto">
  <name>Task 1: Add Link in Bio schema models</name>
  <files>prisma/schema.prisma</files>
  <action>
Add two new models for link-in-bio pages:

1. **LinkBioPage** - The landing page configuration
```prisma
model LinkBioPage {
  id              String    @id @default(cuid())
  userId          String

  // Page identity
  slug            String    @unique  // URL-friendly identifier
  title           String              // Display name
  bio             String?   @db.Text  // Short bio/description

  // Profile
  avatarUrl       String?
  coverUrl        String?

  // Theme settings
  theme           String    @default("default") // 'default', 'minimal', 'gradient', 'dark', 'neon'
  primaryColor    String    @default("#06b6d4") // Cyan-500
  backgroundColor String    @default("#0f172a") // Gray-900
  textColor       String    @default("#ffffff")
  buttonStyle     String    @default("rounded") // 'rounded', 'pill', 'square'

  // Social links (JSON array of {platform, url})
  socialLinks     Json      @default("[]")

  // Settings
  isPublished     Boolean   @default(false)
  showBranding    Boolean   @default(true) // "Powered by Synthex"

  // Analytics
  totalViews      Int       @default(0)
  totalClicks     Int       @default(0)

  createdAt       DateTime  @default(now())
  updatedAt       DateTime  @updatedAt

  user            User      @relation(fields: [userId], references: [id], onDelete: Cascade)
  links           LinkBioLink[]

  @@index([userId])
  @@index([slug])
}
```

2. **LinkBioLink** - Individual links on the page
```prisma
model LinkBioLink {
  id              String    @id @default(cuid())
  pageId          String

  // Link content
  title           String
  url             String
  description     String?
  iconUrl         String?   // Custom icon
  iconType        String?   // 'emoji', 'image', 'lucide'
  iconValue       String?   // Emoji char, image URL, or lucide icon name

  // Display
  order           Int       @default(0)
  isVisible       Boolean   @default(true)
  isHighlighted   Boolean   @default(false)

  // Analytics
  clickCount      Int       @default(0)

  createdAt       DateTime  @default(now())
  updatedAt       DateTime  @updatedAt

  page            LinkBioPage @relation(fields: [pageId], references: [id], onDelete: Cascade)

  @@index([pageId])
  @@index([pageId, order])
}
```

3. Add relation to User model:
```prisma
// In User model, add:
linkBioPages      LinkBioPage[]
```

Run `npx prisma validate` after adding models.
