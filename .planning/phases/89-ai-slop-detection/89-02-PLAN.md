<tasks>
<task id="1" name="Create app/api/quality/audit/route.ts">
POST route. Full quality audit — returns humanness score, slop scan, and full content score.

Body schema: `{ text: string, save?: boolean }`
Validation: text must be non-empty string (no minimum — gate applies to short content too)
Auth: getUserIdFromRequest() — 401 if not authenticated

Behavior:
1. Call scoreHumanness(text) from lib/quality/humanness-scorer.ts
2. Also run content-scorer score() for the full 6-dimension breakdown
3. If save=true: create ContentQualityAudit in DB with results
4. Return: { humanness: HumannessResult, contentScore: ScoreResult, auditId?: string }

Rate limiting: 30/min
Feature gating: check qualityAudits feature key (add to feature-limits in task 2)
</task>

<task id="2" name="Create app/api/quality/gate/route.ts">
POST route. Lightweight gate check — returns pass/fail + reason. No DB persistence.
Purpose: called by publishing/scheduling components before allowing content to publish.

Body schema: `{ text: string, threshold?: number }` (threshold defaults to 60)
Auth: getUserIdFromRequest() — 401 if not authenticated

Behavior:
1. Call scoreHumanness(text, threshold)
2. Return: { passes: boolean, score: number, grade: string, blockingIssues: string[] }
   where blockingIssues = result.issues filtered to errors only (slopDensity > 3, error-severity slop)

Rate limiting: 60/min (higher — lightweight, called before every publish)
No feature gate — this is available on all plans (safety feature)
</task>

<task id="3" name="Extend lib/geo/feature-limits.ts with quality feature key">
Add to GEOFeatureLimits interface (after slopScans):
```typescript
  qualityAudits: number;  // Full quality audits per month (includes DB save)
```

Plan limits:
- free: qualityAudits: 3
- pro: qualityAudits: 50
- growth: qualityAudits: -1
- scale: qualityAudits: -1
- professional (alias): qualityAudits: 50
- business (alias): qualityAudits: -1
- custom: qualityAudits: -1

Add to FEATURE_INFO:
```typescript
  qualityAudits: {
    label: 'Content Quality Audits',
    description: 'Full humanness scoring and AI slop detection with saved audit history',
  },
```
</task>
