# Scratchpad

Temporary working space for Claude Code sessions. Use this directory for:

- In-progress research notes
- Partial outputs that need review
- Session context dumps when context runs low
- Draft plans before committing to `.planning/`

**Convention:** Prefix files with the date (`2026-03-01-topic.md`) for easy cleanup.

**Cleanup:** Files here are ephemeral. Delete anything older than 7 days.
