# 🔑 API Keys Setup Guide for SYNTHEX Users

## Why Do You Need Your Own API Keys?

SYNTHEX uses a **"Bring Your Own Keys"** model, which means:
- ✅ **Lower costs** - You pay AI providers directly (typically $1-10/month)
- ✅ **Full control** - Monitor and manage your own usage
- ✅ **Privacy** - Your data stays between you and the AI providers
- ✅ **No usage limits** - Use as much as you need

## Required API Keys

### 1. 🌟 OpenRouter API Key (Required)
**What it is:** Access to 50+ AI models including GPT-4, Claude, Gemini, and more.

**How to get it:**
1. Visit [openrouter.ai](https://openrouter.ai/)
2. Sign up for a free account
3. Get $5 in free credits to start
4. Go to [Keys section](https://openrouter.ai/keys)
5. Create a new API key
6. Copy the key (starts with `sk-or-v1-`)

**Typical costs:**
- Social media posts: $0.001 - $0.005 each
- Blog articles: $0.01 - $0.05 each
- Content variations: $0.002 - $0.008 each

### 2. 🤖 Anthropic API Key (Optional)
**What it is:** Direct access to Claude models with better rates.

**How to get it:**
1. Visit [console.anthropic.com](https://console.anthropic.com/)
2. Create an account
3. Get $5 in free credits
4. Generate an API key
5. Copy the key (starts with `sk-ant-api`)

**When to use:** If you prefer Claude models or want better rates for Claude usage.

## Setting Up Your Keys in SYNTHEX

1. **Login to your SYNTHEX account** at `/dashboard`
2. **Find the "API Keys Setup" section**
3. **Paste your OpenRouter key** in the first field
4. **Paste your Anthropic key** (optional) in the second field
5. **Click "Save API Keys Securely"**

Your keys are encrypted and stored securely. We never see your actual keys.

## Security & Privacy

- 🔐 **Encrypted Storage** - Keys are encrypted with AES-256-CBC
- 🚫 **No Access** - SYNTHEX developers cannot see your keys
- 🛡️ **Secure Transmission** - All requests use HTTPS
- 🔄 **You Control** - Add, update, or remove keys anytime

## Troubleshooting

### ❌ "API key not configured"
- Make sure you've added your OpenRouter API key
- Check that the key starts with `sk-or-v1-`
- Verify you have credits in your OpenRouter account

### ❌ "Invalid API key format"
- OpenRouter keys: Must start with `sk-or-v1-`
- Anthropic keys: Must start with `sk-ant-api`
- Make sure there are no extra spaces

### ❌ "Rate limit exceeded"
- You've hit your usage limits
- Add more credits to your account
- Wait for the rate limit to reset

### ❌ "Insufficient credits"
- Your account balance is low
- Add credits to your OpenRouter/Anthropic account
- Monitor usage in the dashboard
