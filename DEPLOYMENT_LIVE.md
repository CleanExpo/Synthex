## 🚨 Important Notes

### Database Connection
If you see database errors, you need to:
1. Set up Supabase or PostgreSQL
2. Add `DATABASE_URL` to Vercel environment variables
3. Run migrations: `npx prisma migrate deploy`

### Demo Mode
The platform works in demo mode without a database:
- Login: demo@synthex.com
- Password: demo123

### AI Features
To enable AI content generation:
1. Sign up at openrouter.ai
2. Add credits ($5 minimum)
3. Add `OPENROUTER_API_KEY` to Vercel

### Email Service
To enable email verification:
1. Choose provider (Resend recommended)
2. Get API key
3. Add to Vercel environment variables
